virtual void AstarAIFollow.ReachedEndOfPath ( ) [virtual]

Called when the AI reached the end of path. This will be called once for every path it completes, so if you have a really fast repath rate it will call this function often if when it stands on the end point.

virtual void AstarAIFollow.Repath ( ) [virtual]

Recalculates the path to #target. Queries a path request to the Seeker, the path will not be calculated instantly, but will be put on a queue and calculated as fast as possible. It will wait if the current path request by this seeker has not been completed yet.

bool AstarAIFollow.canMove = true

Can it move. Enables or disables movement and rotation

bool AstarAIFollow.canSearch = true

Should paths be searched for. Setting this to false will make the AI not search for paths anymore, can save some CPU cycles. It will check every repathRate seconds if it should start to search for paths again.

Note:
It will not cancel paths which are currently being calculated
